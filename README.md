# PaperTS Framework

A TypeScript framework for writing Minecraft Paper plugins.

Built on top of [PaperTS](https://github.com/MetlHedd/PaperTS) by [@MetlHedd](https://github.com/MetlHedd).

## Features

- **Full TypeScript Support** - Complete type definitions for Bukkit API
- **Java-like Decorators** - `@EventHandler`, `@Command` just like Bukkit/Spigot
- **Ready-to-use Enums** - EntityType, PotionEffectType, Sound, Material

## Requirements

- [PaperTS](https://github.com/MetlHedd/PaperTS) plugin installed on your Paper server
- Node.js & npm for building
